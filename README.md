# Daily Blog Collector 📰

Automatically collects blog posts from selected sources every day at 6:00 AM.

## What it does
- Runs daily via GitHub Actions
- Scrapes RSS feeds from chosen blogs
- Saves results as JSON in the outputs/ folder
- Organized by topic

## Topics Covered
- Marketing
- Design
- Tech
- Digital Cinema
- AdSense

## Blogs Collected
| Blog | Topic |
|------|-------|
| HubSpot Marketing | Marketing |
| Neil Patel | Marketing |
| Smashing Magazine | Design |
| Creative Bloq | Design |
| TechCrunch | Tech |
| The Verge | Tech |

## Output
Results are saved daily to `outputs/latest.json` and `outputs/posts_YYYY-MM-DD.json`

## Schedule
Every day at 6:00 AM Mexico City time
